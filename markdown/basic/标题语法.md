# Markdown 标题语法

要创建标题，请在单词或短语前面添加井号 (#) 。
"\#"的数量代表了标题的级别。

例如，添加三个 # 表示创建一个三级标题

|Markdown语法|HTML|效果|
|---|---|---|
|\### My Header|\<h3>My Header\</h3>|<h3>My Header</h3>|


## 可选语法

还可以在文本下方添加任意数量的 **== 号来标识一级标题**，或者 **-- 号来标识二级标题**。

Heading level 1
\===============

Heading level 2 
\---------------

## 最佳实践

为了兼容考虑，请用一个空格在 # 和标题之间进行分隔。
